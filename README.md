What is this?
---------------

A kanban board. Implemented with node.js. 
image(kanban.png)

Installation
------------

Install [node.js](http://nodejs.org/)  
Install [node package manager](http://npmjs.org/) 

Install the following node.js plugins:
[jade](http://jade-lang.com/)  
[express](http://expressjs.com/)
